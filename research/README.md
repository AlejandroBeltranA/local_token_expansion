# Research Tracks

This directory contains public research material that informed the maintained LTE product surface but is not itself the main entrypoint for new users.

## Structure

- `propensity/`
  - exploratory work on long-horizon novelty collapse, repetition, and usefulness decay
- `drift_v0/`
  - trigger, intervention, and accountability-oriented reliability research
- `papers/`
  - manuscript and figure-generation assets tied to research tracks

## Status

These tracks are useful and public, but they should be read as supporting research rather than the shortest path to understanding the current product.
