# Propensity Research Track

This directory contains the earlier exploratory work that led into the current LTE product framing.

## Purpose

The propensity work explored when model responses stop being useful over longer runs, especially through:

- novelty collapse,
- repeated-answer behavior,
- truncation pressure,
- format adherence breakdown.

## Status

This track remains useful as research history and as a source of ideas for future LTE triggers.

It is not the maintained product surface.
